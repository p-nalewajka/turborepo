## Usage

First, import the component:

```js
import { Accordion } from 'spoton-lib';
```

Accordion has 2 required props: title and children.

[Check out its implementation example in Accordion Examples.](?path=/docs/nexus-components-accordions-accordion-examples--single-accordion)
